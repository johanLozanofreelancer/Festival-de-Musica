# Festival de Música Rock & EDM
 [Aquí el link:](https://bcn-rockfest-2026.netlify.app/)

Proyecto de práctica para el desarrollo de una landing page optimizada, enfocada en la automatización de tareas y maquetación avanzada.

## 🚀 Tecnologías utilizadas
* **HTML5** (Semántico)
* **SASS** (Arquitectura modular)
* **Gulp** (Automatización de flujo de trabajo)
* **JavaScript** (Galería dinámica y efectos de scroll)
* **WebP / AVIF** (Optimización de imágenes de última generación)

## 🛠️ Automatización con Gulp
En este proyecto utilicé Gulp para optimizar el despliegue:
* Compilación de SASS a CSS.
* Minificación de código para mejorar el rendimiento.
* Conversión automática de imágenes a formatos ligeros (WebP/Avif).

## 💻 Instalación
Si deseas probar este proyecto localmente:
1. Clona el repositorio.
2. Ejecuta `npm install` para instalar las dependencias.
3. Ejecuta `npx gulp` para iniciar el entorno de desarrollo.de venta de boletos para un fesitival de musica, encontraras detalle de los artistas que se presentan, cronograma y precios del evento
